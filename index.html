blogger.relatedposts

Blogger plugin for createing related post list.

View the Project on GitHub garafu/blogger.relatedposts

This plugin generate related posts list atuomaticaly according to the label data.

Features

Main feature is followin.

  • Generate related posts list automaticaly.

Usage

This plugin requires "google feed api". so, include https://www.google.com/jsapi and load feeds. Configuration of this plugin is following.

var RELATEDPOSTS_SETTINGS = {
    BlogUrl: 'garafu.blogspot.jp',
    MaxResults: 5,
    TitleVisible: true,
    ThumbnailVisible: true,
    SnippetVisible: true,
    NoThumbnailImageUrl: 'http://4.bp.blogspot.com/-9mCCaXTpn9I/VN4hDMsFXlI/AAAAAAAAC6U/i84N2Ng37Ik/s1600/noimage.png',
    NoRelatedPostsMessage: 'No Related Posts...',
    LabelElementId: '',
    OutputElementId: ''
};
